## Available Scripts

In the project directory, you can run:

***step 1***
```
$ sudo docker network create -d bridge app-network
```

***step 2***
```
$ sudo docker-compose build --no-cache
```

***step 3***
```
$ sudo docker-compose up -d --no-deps
```
